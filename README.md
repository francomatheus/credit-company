# Credit-Company

Essa API utiliza Spring Boot + H2 + Spring Security + OAuth2 + Swagger e Lombok.


### Estrutura da API 

- config
- controller
- model
    - domain
    - DTO
    - entity
    - enums
    - form
- repository
- security
- service
    - impl

### Funcionalidades

1. Login
2. Usuário
3. Permissões
4. Cadastro
5. Solicitação de Crédito
6. Empréstimo
7. Status das Solicitações
